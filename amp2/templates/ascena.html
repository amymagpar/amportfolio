Design system and components

A Design System and components had to be created for both sites with detailed annotations in order to allow off-shore dev teams to build using the correct spacing and design. Using components also allowed us to redesign for ADA compliance by changing colors and spacing quickly throughout all of the pages.

ADA compliance

A thorough ADA review of the new designs showed that we needed to incorporate requirements into our new Design System. Things like contrast, using the same icons for different actions, and outlining colors for products were some of the changes we made.
